# Pong
A simple  pong game using pygame


## Built with
- [python](https://www.python.org/)
- [pygame](https://www.pygame.org/)
# Usage
- Run main.py and play


# Installation

- Prepare a venv,
for example using conda,

 `conda create -n new_venvname python=3.8 anaconda`
- install pygame

 `pip install pygame`
